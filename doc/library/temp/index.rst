Temporary Frameworks
====================
.. currentmodule:: versile.temp

.. warning::

    These are temporary frameworks which are not permanently part of
    :term:`VPy` and will be replaced in later versions.
    
:mod:`versile.temp` contains temporary frameworks intended to be
replaced later with other mechanisms. It provides simple draft
implementations of concepts needed for a complete service
infrastructure, but which are not yet fully implemented.
