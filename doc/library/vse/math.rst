Math
====
.. currentmodule:: versile.vse.math

The module :mod:`versile.vse.math` includes :term:`VSE` classes for
higher order mathematics.

The following :term:`VSE` types are registered with
:class:`VMathModule`\ .

+-------------------------------+--------------------------------------+
| Type                          | Description                          | 
+===============================+======================================+
| :class:`VPrefixedUnit`        | Unit scaled by a prefix              |
+-------------------------------+--------------------------------------+
| :class:`VDimensionalQuantity` | Dimensional quantity                 |
+-------------------------------+--------------------------------------+
