Time
====
.. currentmodule:: versile.vse.time

The module :mod:`versile.vse.time` includes :term:`VSE` classes for
handling time references.

The following :term:`VSE` types are registered with
:class:`VTimeModule`\ .

+-------------------------+--------------------------------------------+
| Type                    | Description                                | 
+=========================+============================================+
| :class:`VUTCTime`       | Universal Coordinated Time time reference  |
+-------------------------+--------------------------------------------+
